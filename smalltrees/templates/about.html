About this project
Scope

This website presents an algebraic classification of small tree and network models in phylogenetics. It currently hosts data on twentysix trees, paired with five different model types. These are listed in the table on the landing page. Mathematical details can be found on individual subpages for each tree and model. The content displayed on this website is work in progress! You can see its current version in the upper left hand corner: currently, version 0.1. We focus on providing thorough computational proof for every result we display. To achieve this, the website is supplemented by the AlgebraicPhylogenetics.jl software. The code is part of the AlgebraicStatistics.jl package in the OSCAR project, written in Julia. OSCAR provides serialization which, in the coming version 0.2, we use to make our data available for immediate embedding into your own code. You can download all our data under this link, in the .mrdi format. Never heard about algebraic phylogenetics? The page Notions and notation gives a mathematical introduction to the topic, and presents code snippets in a running example.
