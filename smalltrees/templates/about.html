About this project

Scope

This website presents an algebraic classification of small tree and network models in phylogenetics. It currently hosts data on twentysix trees, paired with five different model types. These are listed in the table on the landing page. Mathematical details can be found on individual subpages for each tree and model. We focus on providing thorough computational proof for every result. To achieve this, the website is supplemented by the AlgebraicPhylogenetics.jl software. The code is part of the AlgebraicStatistics.jl package in the OSCAR project, written in Julia. The page Notions and notation gives a mathematical introduction to the topic, and presents code snippets in a running example. You can download all our data under this link, in the .mrdi format.
