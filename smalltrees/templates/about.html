About

Scope

This website presents an algebraic classification of small tree models in phylogenetics. It currently hosts data on twentysix trees, paired with five different model types. For each, we determine algebraic invariants which describe the model's geometry. These are listed in the table on the landing page. Mathematical details can be found on individual subpages for each tree and model. We focus on providing thorough computational proof for every result. To achieve this, the website is supplemented by the AlgebraicPhylogenetics.jl software. The code is part of the AlgebraicStatistics.jl package in the OSCAR project, written in Julia.
